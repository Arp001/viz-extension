Semi-Circular Gauge Chart

Tableau Dashboard Extension

What is this?

A configurable speedometer-style gauge extension for Tableau dashboards. It reads data from your worksheets and renders a semi-circular gauge with colored range bands, an animated needle, tooltips, and click-to-filter interactivity.

How to use

  • In Tableau Desktop, add an Extension zone to your dashboard.
  • Load the .trex manifest file that points to this URL.
  • Right-click the extension → Configure to select your worksheet, measure, and visual settings.

Extension entry point

The main extension page is gauge.html. The .trex manifest should point to that URL.
