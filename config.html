⚙ Gauge Configuration

The gauge's maximum scale value is computed from this field using the selected aggregation, and updates dynamically with filters.
The gauge maximum is computed as Goal × Multiplier. Enter a number (1.5) or a percentage (150%) — both set Max to 1.5× the Goal. The scale grows proportionally as the Goal changes. Requires a Goal field (below).
Auto-Detect: If min=0 and max≤1, treats data as 0–1 ratios and converts to percentages.
🎯 Goal Field

A single shared reference value computed from a worksheet field. It can be used as a range start boundary (see “% of Goal” and “Goal Field Value” below) and is validated against the Max value.

🎨 Ranges & Colors

Define color bands on the gauge. Each range has a label, color and a start boundary. A range ends where the next one starts; the last range ends at Max. The first range's start is whatever you set.

Presets:
Use Smooth Color Gradients
When enabled, color bands blend smoothly into each other instead of using hard color stops.
